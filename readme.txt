此开源项目为锐浪软件Grid++Report6的HTML5报表的源码，任何人使用本源码不受任何版权限制，可以用于任何用途。

几点重要说明：
1、报表模板文件(.grf)通常用Grid++Report6报表设计器程序制作，安装Grid++Report6开发者安装包可以获取到此程序。
2、开源部分的代码不能解析Grid++Report6注册加密的报表模板。源码中与“isWR”变量不应该为“真(true)”值，有关的代码也不应该被执行到。
3、欢迎大家将改进后的代码提交给锐浪软件，以便锐浪软件考虑将您增强的功能合并到后续版本中。
4、Grid++Report6的HTML5报表最终是将相关的源码合并为一个.js文件，然后用Uglifyjs进行压缩。
5、Grid++Report6开发者安装包安装后的相关例子可以用于测试。
